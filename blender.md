#  概述

## 3D建模基本流程

1. 建模(Modeling)
2. 布光(Lighting)
3. 材质(texturing)
4. 渲染(rendering)

## 游标

> 游标为布局中心红色的圆点，游标在哪里，新建的物体就会创建在哪里；

- 作用
  1. 可以作为旋转轴心点
  2. 可以快速定位物体位置

## 原点

> 物体上黄色的点，记录了物体的信息

- 作用
  1. 通过移动物体的原点，可以移动物体缩放的位置

# 十大建模操作

- **挤出**

  快捷键为E,可以选择一个点挤出一条边，选择一条边挤出一个面，选择一个面，挤出一个立方体

  注意:

  - 挤出实际上包括两个操作:挤出 + 移动，当挤出操作生效后，第一次撤销动作只能撤销移动，挤出的物体仍然在原地。

- 内插(向内挤出)

  快捷键I,在选中的面中新增一个内插面

# 快捷键

- 旋转观察视角

  ```
  鼠标中键
  ```

- 快速布置物体

  ```
  Shift + a
  ```

- 平移视图

  ```
  Shift + 鼠标中键
  ```

- 缩放视图

  ```
  滚动鼠标中间滚轮
  ```

- 删除物体

  ```
  X
  ```

- 移动物体

  ```
  G
  ```

  > 注意:
  >
  > - 按下G后，可以任意移动物体，按住X情况下移动物体，会沿x轴移动，按住y，沿y轴移动，按住z，沿z轴移动

- 缩放物体

  ```
  S
  ```

- 旋转物体

  ```
  R
  ```

- 隐藏物体

  ```
  H
  ```

- 显示全部隐藏物体

  ```
  alt + H
  ```

- 隐藏所有未选中的物体

  ```
  Shift + H
  ```

- 复制物体

  ```
  Shift + D
  ```

- 刷选工具

  ```
  C
  ```

- 切换视图

  ```
  ~或小键盘数字
  ```

- 最大化当前面板

  ```
  Ctrl + Blank
  ```

- 快速移动游标

  ```
  shift + 右键
  ```

- 移动游标或选中项到特殊位置

  ```
  shift + S
  ```

- 游标回到世界原点

  ```
  Shift + C
  ```

- 清除选中物体的移动

  ```
  alt + G  
  ```

- 吸附网格移动

  > 在移动模式下，按住ctrl移动，会吸附网格移动
  
- 进入编辑模式/退出编辑模式

  ```
  TAB
  ```

  > 注意:
  >
  > - 打开设置中的Tab键调用饼菜单，长按Tab键后，可以随意切换模式

- 切换点线面模式

  ```
  在编辑模式下，
  1，2，3切换点线面模式
  ```

  > 注意:
  >
  > - 按住Shfit，通过左上角点线面模式按钮可以多选模式

- 反选其他点线面

  ```
  编辑模式下，
  CTRL + I
  ```

- 选择最短的连续点线面

  ```
  在编辑模式下，选择一个起点的点线面
  按住CTRL的情况下，选择另一个终点
  ```

- 选择所有相连的点线面

  ```
  在编辑模式下，选择一个点线面
  Ctrl + L
  ```

- 选择一圈点线面

  ```
  按住ALT，
  双击想要选择的那一圈的点线面的其中一个
  ```

- 方向操作

  ```
  ALT + N
  ```

- 挤出

  ```
  编辑模式下，
  按住E,
  可以CTRL + 右键连续挤出
  ```
