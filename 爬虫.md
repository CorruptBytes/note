# 概述

> 网络爬虫是一种按照一定规则，自动地抓取万维网信息的程序或者脚本

## 实现原理

> 爬虫一般分为两步。

1. 向服务器发起请求获取响应数据
2. 从响应数据中解析出需要的数据

# 发送请求

> 发送请求的方式多种多样，根据自己的需求，选择合适的语言与模块

## Urllib.request

> 这是python中一个用于发送http请求的模块,数据Python标准库，无需下载。

- 基础使用

```python
from urllib.request import urlopen
resp = urlopen("https://www.example.com")
html = resp.read().decode("utf-8")
print(html)
```

### 常用函数

- urlopen

> 向指定url发送请求，并返回一个HTTPResponse对象，该对象封装了服务器的响应信息，通过该对象，可以获取 HTTP、HTTPS、FTP 等协议的响应内容、状态码、响应头等。

```python
def urlopen(url: str, data: Optional[bytes] = None, timeout: Optional[float] = None) -> HTTPResponse
```

|  参数   |                             解释                             |
| :-----: | :----------------------------------------------------------: |
|   url   |              请求的url地址，必填，类型为字符串               |
|  data   |  发送post请求时传递的数据，可选，类型为字节数组，默认为none  |
| timeout | 指定请求的超时时间，单位是秒，可选，类型为float，默认none为没有超时限制 |

## re

> re是 Python 中用于支持正则表达式操作的标准库模块，提供了对正则表达式的匹配、搜索、替换和分割等功能。

### 常用函数

- findall

  > 匹配一个字符串中所有符合正则表达式规则的片段，返回一个列表

  ```python
  def findall(pattern:str,string:str,flags:boolean) -> list
  ```

- finditer

  > 匹配字符出啊中所有符合正则表达式规则的片段，返回一个迭代器

  ```
  def finditer(pattern:str,string:str,flags:boolean) ->iterator
  ```

  > 注意:
  >
  > - 迭代器中是re.Match对象，通过group()得到内容

- search

  > 匹配第一个符合正则表达式规则的片段，返回一个Match对象

  ```
  search(pattern,string,flags) -> Match
  ```

- match

  > 从头开始匹配第一个符合正则表达式规则的片段，返回一个Match对象

  ```
  match(pattern,string,flags) -> Match
  ```

  > 注意:
  >
  > - 实际效果就是在正则表达式开头加`^`

- compile

  > 预加载正则表达式，接收一个正则表达式，返回一个符合该规则的Pattern对象

  ```
  compile(pattern) -> Pattern
  ```

  > 注意:
  >
  > - Pattern对象可以调用所有正则匹配方法

# 小知识

## regex（正则表达式）

> 正则表达式(Regular Expression)是一种强大的文本匹配和操作工具，由一系列字符和特殊符号组成，用于描述和匹配特定的文本模式。

- 语法

> 正则表达式由元字符、量词和普通字符等部分组成，形成用于匹配文本的字符串模式。

### 元字符

> 元字符是正则表达式中具有特殊含义的字符，它们用于定义匹配规则，控制匹配的方式。

| 字符 |                             含义                             | 示例 |
| :--: | :----------------------------------------------------------: | :--: |
|  .   |                  匹配任意单个字符(除换行符)                  |      |
|  ^   | 出现在正则表达式开头时，表示匹配字符串的开头;出现在[]中表示取反，匹配不在字符集中的字符 |      |
|  $   |                       匹配字符串的结尾                       |      |
|  []  |          定义一个字符集，匹配字符集内的任何单个字符          |      |
|  \|  |           逻辑或，逻辑“或”，匹配左侧或右侧的表达式           |      |
|  ()  |           分组，用于将多个字符作为一个单位进行匹配           |      |
|  \   |                  转义字符，用于匹配特殊字符                  |      |
|  \w  |                    匹配字母或数字或下划线                    |      |
|  \s  |                       匹配任意的空白符                       |      |
|  \d  |                           匹配数字                           |      |
|  \n  |                        匹配一个换行符                        |      |
|  \t  |                        匹配一个制表符                        |      |
|  \W  |                   匹配非字母或数字或下划线                   |      |
|  \D  |                          匹配非数字                          |      |
|  \S  |                         匹配非空白符                         |      |

### 量词

> 量词用于指定字符或子表达式出现的次数。它控制匹配的数量范围。

| 字符  |       含义       |
| :---: | :--------------: |
|   *   | 重复零次或更多次 |
|   +   | 重复一次或更多次 |
|  ？   |  重复零次或一次  |
|  {n}  |     重复n次      |
| {n,}  | 重复n次或更多次  |
| (n,m} |    重复n到m次    |

### 普通字符

> 普通字符指的是没有特殊意义的字符，它们在正则表达式中按照字面意思进行匹配。例如，字母、数字和常见的标点符号都属于普通字符。
