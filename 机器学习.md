# 监督学习

- 回归

  > 回归(regression)是最简单的监督学习任务之一，目标是根据一组输入特征，预测一个或多个连续值作为输出。该输出可以是任意实数，而不是有限的类别。对于回归问题的误差评估通常采用平方误差损失函数。

- 分类

  > 分类（Classification）是监督学习中的一种任务，其目标是通过一组输入特征，预测样本所属的离散类别。最简单的分类问题只预测两类，称为二项分类(binomial classification)。当有两个以上的类别时，这个分类问题称为多项分类(multiclass classification)。对于分类问题的误差评估通常使用交叉熵(cross-entropy)损失函数

- 标记

  > 标记问题是一种特殊的分类问题，目标是通过输入预测一组不相互排斥的类别，又被称为多标签分类(multi-label classification)，类似为文章添加多个相关标签。

- 搜索

  > 搜索任务的目标是在一个复杂输出空间中，根据输入找到最优的输出序列或结构。
  >
  > 常见应用有机器翻译，搜索引擎搜索排序

- 序列学习(Sequence Learning)

  > 序列学习是一种特殊的监督学习任务，其目标是根据具有时序或顺序特征的输入序列，预测对应的输出序列。

## softmax回归

> Softmax 回归（又称多类逻辑回归）是用于 **多分类问题** 的一种**线性模型**，用于将一个输入样本映射到多个可能的类别之一。给定输入特征向量 x,Softmax 回归的输出是一个向量，每个元素表示属于某个类别的**概率**。softmax回归本质上是一个单层神经网络(具有一个输入为n,输出为c的全连接层)。在全连接层的输出后，通过softmax函数将输出映射成一个概率分布。

### 独热编码

> 独热编码(one-hot encoding)用于分类问题中类别之间没有自然顺序的情况。如果类别具有自然顺序(如{婴儿，儿童，青少年}),直接对类别进行简单编码是合理且有意义的({1,2,3})。如果类别不具有自然顺序，就需要使用独热编码。独热编码是一个向量，它的分量和类别一样多。 类别对应的分量设置为1，其他所有分量设置为0(如{(1,0,0),(0,1,0),(0,0,1)})

### softmax运算

> softmax函数能够将未规范化的预测变换为非负数并且总和为1，同时让模型保持 可导的性质。
>
> 尽管softmax是一个非线性函数，但softmax回归的输出仍然由输入特征的仿射变换决定。 因此，softmax回归是一个线性模型。

# 多层感知机

> 多层感知机(Multilayer Perceptron, 简称 **MLP**)是一种由多个线性变换和非线性激活函数交替堆叠而成的前馈神经网络。这里的线性变换是指通过全连接层实现的仿射变换。多层感知机通过为模型添加非线性因素，提高了模型的表达能力。

## 组成

- 输入层(Input Layers)

  > 一个逻辑上的概念是模型对输入张量的处理入口，负责将数据送入网络，不含有可学习参数。实际在代码上并不存在。

- 隐藏层(Hidden Layers)

  > 每个隐藏层包含两部分:

  - 一个 **线性变换（仿射变换）**：即全连接层 y=Wx+b其中 W 和 b 是可学习参数。
  - 一个 **非线性激活函数**：如 ReLU、Sigmoid、Tanh 等，用于引入非线性。

    **可以将多个隐藏层堆叠使得网络具备更强的表示能力。**通常选择2的若干次幂作为层的宽度

- 输出层(Output Layers)

  > 通常也是一个「线性变换 + 激活函数」的组合。可根据应用场景来决定是否在输出层加入激活函数

## 激活函数

> 激活函数(activation function)通过计算仿射变换后的数据是否能激活神经元，将输入信号转换为输出的可微运算(即输出可对输入求导)。大多数激活函数都是非线性的。

# 卷积神经网络

> convolutional neural networks，CNN。是包含卷积层的一类特殊的神经网络

- 卷积神经网络基于以下原则
  - *平移不变性*（translation invariance）：不管检测对象出现在图像中的哪个位置，神经网络的前面几层应该对相同的图像区域具有相似的反应，即为“平移不变性”。
  - *局部性*（locality）：神经网络的前面几层应该只探索输入图像中的局部区域，而不过度在意图像中相隔较远区域的关系，这就是“局部性”原则。最终，可以聚合这些局部特征，以在整个图像级别进行预测。

## 从全连接层推导卷积层

> 将全连接的输入与输出从一维推广到二维，即输入特征与输出特征均是一个矩阵。

- 假设输入：矩阵 M×N

  > 每个神经元：需要对 **所有输入元素** 加权求和，所以它的权重是一个 M×N的矩阵。

- 要求输出:矩阵

  > 即矩阵需要有M×N个神经元生成M×N个新特征。所以此时权重矩阵应该为四维(M,N,M,N)

  ![image-20250811221905446](D:\笔记\图片\image-20250811221905446.png)

  - X表示输入特征，H表示新特征，i,j为新特征在矩阵中的位置
  - U表示偏置矩阵，一个神经元具有一个偏置
  - 最后表示神经元的权重矩阵与输入特征进行加权求和，再加上偏置，最终生成一个新特征
  - W到V只是形式上的变化，方便后续变换

- 平移不变性意味着检测对象在输入X中的平移，应该仅导致隐藏表示H中的平移。也就是说，V和U实际上不依赖于(i,j)的值，并且U是一个常数，

  ![image-20250811222412597](D:\笔记\图片\image-20250811222412597.png)

  ![image-20250811222426041](D:\笔记\图片\image-20250811222426041.png)

  - 这就是卷积，此时四维权重矩阵退化为二维，我们使用二维矩阵V对位置(i,j)附近的元素(i+a,j+b)进行加权得到H。

- 根据局部性原则，我们不应该在意距(i,j)很远的地方。这意味着，可以将`|a|>∆`或`|b|>∆`的范围之外，可将V的元素置0。

  ![image-20250811223202818](D:\笔记\图片\image-20250811223202818.png)

  - 这就形成了一个卷积层(convolutional layer)
  - 矩阵V被称为卷积核（convolution kernel）或者滤波器（filter）,也可简单的称其为该卷积层的权重
  - 相比于MLP中庞大的参数量，卷积神经网络（CNN）通过权重共享和局部连接显著减少了参数数量，同时通常保持输入和隐藏层表示的空间维度不变。
  - 这种设计带来的代价是，卷积层天生具有平移不变性，即相同的特征无论出现在输入的哪个位置，卷积层都会用相同的权重进行检测。然而，某些任务中，特征的位置具有重要语义意义，平移不变性在这时可能成为限制。
  - 此外，卷积层的每个激活值仅利用局部感受野内的信息进行计算，意味着网络对全局信息的感知需要通过更深的层级逐步累积。
  
- 在日常使用中卷积层的输入图像通常是三维的，即(height,weight,channel),因此此时卷积核应该为三维的，才能做到对三维图像进行加权求和。同时我们还要求，卷积层的输出也要保持三维,即`(height′,weight′,channel′)`。故需要`channel′`个卷积核堆叠形成一个四维卷积核才能满足要求

## 互相关运算

> 严格来说，卷积层的叫法是错误的，因为	在卷积层中，输入张量与卷积核之间的运算是互相关运算，而非卷积运算

## 感受野与特征映射

> 卷积层有时被称为*特征映射*（feature map），因为它可以被视为一个输入映射到下一层的空间维度的转换器。 在卷积神经网络中，对于某一层的任意元素x，其*感受野*（receptive field）是指在前向传播期间可能影响x计算的所有元素（来自所有先前层）。

## 池化层

> 池化层又称汇聚层(pooling layer)，通常配合卷积层使用，为了降低卷积层对位置的敏感性，同时降低对空间降采样表示的敏感性。

- 原理

  > 与卷积层类似，汇聚层运算符由一个固定形状的窗口组成(这个窗口通常称为汇聚窗口)，该窗口根据其步幅大小在输入的所有区域上滑动并在每个位置计算出一个输出。不过，池化层不含参数，通常会计算汇聚窗口中所有元素的最大值或平均值。这些操作分别称为最大池化层（maximum pooling）和平均池化层(average pooling）。

  ![image-20250812163803614](D:\笔记\图片\image-20250812163803614.png)

  > 汇聚窗口形状为`p *q`的汇聚层称为`p *q`汇聚层，汇聚操作称为`p *q`汇聚。

- 与卷积层一样，池化层也可以设置步长与填充来获得所需的输出形状

- 池化层不会改变输入的通道数，即他会在每个通道上单独运算

## VGG

> VGG网络是一种深度卷积神经网络，主要由VGG块和全连接层组成。是首个以块作为功能单元的神经网络。在VGG网络中，每个块由`卷积层 + 激活函数 + 池化层`组成

### 核心思想

1. 使用小卷积核堆叠代替大卷积核

   > 通过两个或三个`3x3`卷积层堆叠代替原本可能使用`7x7`或`5x5`卷积层。不但减少了参数，增强了模型的非线性(这增强了模型提取复杂特征的能力)，还能获得与大卷积层相同的感受野。

2. 层次化特征提取

   > 每隔几层卷积后用 **2×2 最大池化**降采样，逐步提取高级特征

3. 深化网络层次

   > 通过堆叠更多卷积层，增强网络的表达能力。常用的VGG网络变体如VGG16，VGG19，分别有16和19个卷积层。

### 特点

**优点：**

- 结构简单，易理解和实现
- 小卷积核堆叠降低参数量，增加非线性
- 可迁移性强，适合做 backbone

**缺点:**

- 参数量仍然很大（尤其 FC 层）
- 训练深层网络时容易梯度消失

### 基本结构

> VGG网络主要由VGG块和全连接层组成。前面使用VGG块提取特征，后面通过全连接层进行分类。

#### VGG块

> 由卷积层堆叠 + 激活函数（ReLU） + 池化层组成。一般各种VGG网络的变体会在网络中连续接入5个VGG块

```
[Conv3x3 -> ReLU] × n -> MaxPool2x2
```

- 卷积层带填充以保持分辨率
- 每个卷积层后跟一个激活函数(通常使用ReLU)
- 通常使用`2x2`最大池化层。
- 不同VGG网络变体中VGG块的卷积层个数可能不同，一般为2~4层
- 每个卷积块内卷积层通道数一致
- 每经过一个VGG块，通道数通常翻倍或增加

```python
def vgg_block(num_convs, in_channels, out_channels):
    layers = []
    for _ in range(num_convs):
        layers.append(nn.Conv2d(in_channels, out_channels,
                                kernel_size=3, padding=1))
        layers.append(nn.ReLU())
        in_channels = out_channels
    layers.append(nn.MaxPool2d(kernel_size=2,stride=2))
    return nn.Sequential(*layers)
```

#### 全连接层

> 通常将积块输出拉平成一维后，接 3 个全连接层。

```
FC1: 4096 → FC2: 4096 → FC3: 输出类别数
```

> 全连接层部分可以视任务调整或去掉

### VGG网络族

> VGG网络族中的各种网络均是由VGG块与全连接层构成，只是各变体中卷积层的个数不同

| 网络  | 卷积层数 | 卷积块堆叠  | 全连接层 |
| ----- | -------- | ----------- | -------- |
| VGG11 | 8        | [1,1,2,2,2] | 3        |
| VGG13 | 10       | [2,2,2,2,2] | 3        |
| VGG16 | 13       | [2,2,3,3,3] | 3        |
| VGG19 | 16       | [2,2,4,4,4] | 3        |

- VGG16

  > 最经典、应用最广的 VGG 变体，总共有 16 个带参数层（13 个卷积层 + 3 个全连接层），用在大小为`224x224`的图片上

  | Block   | 卷积层数量 | 输出通道数 | 说明                         |
  | ------- | ---------- | ---------- | ---------------------------- |
  | Block 1 | 2          | 64         | (Conv3x3 + ReLU)×2 + MaxPool |
  | Block 2 | 2          | 128        | (Conv3x3 + ReLU)×2 + MaxPool |
  | Block 3 | 3          | 256        | (Conv3x3 + ReLU)×3 + MaxPool |
  | Block 4 | 3          | 512        | (Conv3x3 + ReLU)×3 + MaxPool |
  | Block 5 | 3          | 512        | (Conv3x3 + ReLU)×3 + MaxPool |
  | 全连接  | 3          | -          | FC4096 → FC4096 → FC1000     |

- VGG19

  > 比 VGG16 更深，多了 3 个卷积层。两者结构相同，只是VGG19卷积块的堆叠更多

## ResNet

> 残差网络(Residual Network),是一种深度卷积神经网络。通过残差块(Residual Network)解决了网络加深导致性能退化的问题

### 核心思想

1. **残差学习（Residual Learning）**

   - 传统网络学习目标映射 H(x)，即直接从输入 `x `映射到输出 `H(x)`。

   - ResNet 改为学习 **残差映射** `F(x)=H(x)−x`，即

     ![image-20250821172449748](D:\笔记\图片\image-20250821172449748.png)

   - 如果最优映射是恒等映射（网络某些层无需改动），残差学习让网络轻松学习 F(x)=0，避免退化问题。

2. **跳跃连接（Skip Connection / Shortcut Connection）**

   - 残差块中，输入直接跳过卷积层与激活层，加到输出上。
   - 这一操作保证梯度可以直接从后面层传到前面层，缓解梯度消失问题。

### 特点

**优点：**

- 能够训练非常深的网络
- 解决深层网络退化问题
- 梯度流更稳定，训练速度快

**缺点：**

- 对轻量化任务可能参数冗余

### 基本结构

> 通常残差网络可以分为三个部分:浅层使用多个残差块进行特征提取与深层建模，深层通过全连接层进行特征整合与分类决策。

#### 前置层(Stem)

> 前置层使用`7x7`卷积层和一个最大池化层

```
7×7 Conv, stride=2 -> MaxPool 3×3, stride=2
```

- **快速降采样**：将输入图像分辨率降低，减少后续计算量
- **初步特征提取**：用大卷积核捕捉较大感受野的低级特征（边缘、纹理等）。
- **准备残差块输入**：生成适合后续残差块处理的 feature map，保证通道数与尺寸匹配。
- 前置层是 ResNet 的“预处理器”，把原始图像转化为适合残差块处理的特征图。
- 通常卷积层输出64个通道
- 适当加入填充，保证输入与输出为2的倍数

#### 残差块(Residual Block)(Conv2_x~Conv5_x)

> 常见的残差块结构有两种，一种用于浅层网络，另一种用于深层网络

- Basic Block（基础残差块，用于浅层网络，如 ResNet-18/34）

  ```
  输入 x
  [Conv3x3 -> BN -> ReLU -> Conv3x3 -> BN] + x -> ReLU
  ```

  - 使用两个`3x3`卷积层
  - 输出=`卷积输出 + 原输入`
  - 如果卷积输出与原输入在通道数上不匹配，可以在原输入后接`1x1`卷积层调整
  - 第一个卷积层通常设置为通道翻倍，步长为2，以使通道数增加、分辨率减半。第二个卷积层通道不变，步长为1，同时原输入要接入`1x1`卷积保证形状匹配

- Bottleneck Block（瓶颈残差块，用于深层网络，如 ResNet-50/101/152）

  ```
  输入 x
  [Conv1x1 -> BN -> ReLU -> Conv3x3 -> BN -> ReLU -> Conv1x1 -> BN] + x -> ReLU
  ```

  - 1x1 卷积通道数降维到1/4输出 → 3x3 卷积通道不变，分辨率减半 → 1x1 卷积通道数乘4倍升维
  - 降低计算量，同时保持深度网络性能


#### 分类层

> 分类层使用全局平均池化层与全连接层

- 全局平均池化层

  > 将每个通道的特征图压缩成一个数（平均值），消除了空间维度，减少参数量，增强模型泛化能力。

- 全连接层

> 残差网络通常在最后接入一层全连接层将所有通道特征整合，用于最终分类或回归任务。

### ResNet网络族

| 网络       | 前置层               | 残差块类型 | 残差块堆叠 (Conv2_x ~ Conv5_x) | 分类层   | 卷积层个数(不包括跳跃连接) |
| ---------- | -------------------- | ---------- | ------------------------------ | -------- | -------------------------- |
| ResNet-18  | 7×7 Conv, MaxPool2x2 | BasicBlock | [2, 2, 2, 2]                   | GAP + FC | 17                         |
| ResNet-34  | 7×7 Conv, MaxPool2x2 | BasicBlock | [3, 4, 6, 3]                   | GAP + FC | 33                         |
| ResNet-50  | 7×7 Conv, MaxPool2x2 | Bottleneck | [3, 4, 6, 3]                   | GAP + FC | 49                         |
| ResNet-101 | 7×7 Conv, MaxPool2x2 | Bottleneck | [3, 4, 23, 3]                  | GAP + FC | 100                        |
| ResNet-152 | 7×7 Conv, MaxPool2x2 | Bottleneck | [3, 8, 36, 3]                  | GAP + FC | 151                        |

> 注意:
>
> - 命名后的数字是卷积层个数+全连接层个数，卷积层不包括跳跃连接的`1x1`卷积层
> - ResNet系列网络通常把卷积层堆叠分成 4 个主要阶段，每个阶段由多个 **残差块（Residual Block）** 组成。每进入新阶段，通道数翻倍，空间分辨率减半

- ResNet-50

  > 最经典、最常用的残差网络，在深度和性能上都比较平衡。

  | 阶段        | 名称            | 输入大小（假设输入 224×224） | 通道数 | 特点                           |
  | ----------- | --------------- | ---------------------------- | ------ | ------------------------------ |
  | **Stage 1** | conv1 + maxpool | 112×112 → 56×56              | 64     | 卷积 stem（不是残差块部分）    |
  | **Stage 2** | conv2_x         | 56×56                        | 256    | 残差块 bottleneck 输出通道 256 |
  | **Stage 3** | conv3_x         | 28×28                        | 512    | 通道翻倍，分辨率减半           |
  | **Stage 4** | conv4_x         | 14×14                        | 1024   | 通道翻倍，分辨率减半           |
  | **Stage 5** | conv5_x         | 7×7                          | 2048   | 通道翻倍，分辨率减半           |

  1. **stem 卷积部分**
     - 7×7 卷积，stride=2，输出通道 = 64
     - 3×3 maxpool，stride=2
        → 输出通道 = 64，分辨率 = 1/4
  2. **Conv2_x 阶段（Stage 1）**
     - 第一个 Bottleneck 块：1×1 conv (64→64)，3×3 conv (64→64)，1×1 conv (64→256)
     - 注意最后的 1×1 卷积把通道数扩展到 **256**
     - 后续 block 保持 256
       → **所以 stage1 的输出通道是 256**

  3. **Conv3_x 阶段（Stage 2）**
     - 第一个 Bottleneck：1×1 conv (256→128)，3×3 conv (128→128)，1×1 conv (128→512)
     - 输出通道变成 **512**，空间分辨率减半
     - 后续 block 维持 512

  4. **Conv4_x 阶段（Stage 3）**
     - Bottleneck：1×1 conv (512→256)，3×3 conv (256→256)，1×1 conv (256→1024)
     - 输出通道变成 **1024**，分辨率再减半

  5. **Conv5_x 阶段（Stage 4）**
     - Bottleneck：1×1 conv (1024→512)，3×3 conv (512→512)，1×1 conv (512→2048)
     - 输出通道 **2048**，分辨率再次减半

  6. **全局平均池化 + 全连接层**
     - 分类输出


## NiN

> 深度卷积神经网络的一种改进架构。

**核心思想**：在传统卷积层后加入 **1×1 卷积**，称为 **MLPConv**，相当于在空间位置上做逐通道的全连接，提升特征表达能力。

**结构**：

- 由多个 **NIN 块**（卷积层 + 1×1 卷积 + 激活）堆叠
- 末尾使用 **全局平均池化（Global Average Pooling）** 代替全连接层进行分类

**特点**：减少参数量，抑制过拟合，对输入空间位置更敏感。

> NiN块的想法是将空间维度中的每个像素视为单个样本，将通道维度视为不同特征（feature）。因此每个像素位置应用一个全连接层，将其通道加权求和，相当于`1x1卷积层`

- 组成

  >  NiN块以一个普通卷积层开始，后面是两个`1x1`的卷积层。这两个卷积层充当带有ReLU激活函数的逐像素全连接层。 第一层的卷积窗口形状通常由用户设置。 随后的卷积窗口形状固定为`1x1`。

  ![image-20250813113912237](D:\笔记\图片\image-20250813113912237.png)

```python
def nin_block(in_channels, out_channels, kernel_size, strides, padding):
    return nn.Sequential(
        nn.Conv2d(in_channels, out_channels, kernel_size, strides, padding),
        nn.ReLU(),
        nn.Conv2d(out_channels, out_channels, kernel_size=1), nn.ReLU(),
        nn.Conv2d(out_channels, out_channels, kernel_size=1), nn.ReLU())
```

- NiN网络

  > 最初的NiN网络在每个NiN块后跟一个最大汇聚层，并且完全取消了全连接层，代替为一个NiN块，其输出通道数等于标签类别的数量。最后放一个全局平均汇聚层（global average pooling layer），生成一个对数几率 （logits）。

  - 取消了容易过拟合的全连接层，显著减少了模型所需参数的数量。然而，在实践中，这种设计有时会增加训练模型的时间。

- VGG：卷积块 + 全连接层，参数多，适合大数据集
- NIN：卷积 + 1×1 卷积 + 全局平均池化，参数少，泛化好

## GoogLeNet

> 在GoogLeNet中，基本的卷积块被称为*Inception块*

![image-20250813120852969](D:\笔记\图片\image-20250813120852969.png)

- Inception块由四条并行路径组成。 前三条路径使用窗口大小为`1x1`,`3x3`,`5x5`的卷积层，从不同空间大小中提取信息。 中间的两条路径在输入上执行`1x1`卷积，以减少通道数，从而降低模型的复杂性。 第四条路径使用`3x3`最大汇聚层，然后使用`1x1`卷积层来改变通道数。 这四条路径都使用合适的填充来使输入与输出的高和宽一致，最后我们将每条线路的输出在通道维度上连结，并构成Inception块的输出。在Inception块中，通常调整的超参数是每层输出通道数。

## ResNet

> 又称为*残差网络*，核心思想是：每个附加层都应该更容易地包含原始函数作为其元素之一。由*残差块*（residual blocks）组成,下图右侧即为一个残差块，在残差块中，输入可通过跨层数据线路更快地向前传播。

![image-20250813203502235](D:\笔记\图片\image-20250813203502235.png)

- 核心思想

  > ResNet的核心思想是把想要学习出的映射改为残差函数。F(x)为残差函数，H(x)为目的映射。因为残差映射在现实中往往更容易优化

  ![image-20250813203613282](D:\笔记\图片\image-20250813203613282.png)

- 残差块组成

  > 残差块里首先有2个有相同输出通道数的`3x3`卷积层。 每个卷积层后接一个批量规范化层和ReLU激活函数。 然后我们通过跨层数据通路，跳过这2个卷积运算，将输入直接加在最后的ReLU激活函数前。 这样的设计要求2个卷积层的输出与输入形状一样，从而使它们可以相加。 如果想改变通道数，就需要引入一个额外的`1x1`卷积层来将输入变换成需要的形状后再做相加运算。

## DenseNet

>  *稠密连接网络*在某种程度上是ResNet的逻辑扩展.

![image-20250813205436231](D:\笔记\图片\image-20250813205436231.png)

- 稠密网络主要由2部分构成：*稠密块*（dense block）和*过渡层*（transition layer）。 前者定义如何连接输入和输出，而后者则控制通道数量，使其不会太复杂。

> 一个*稠密块*由多个卷积块组成，每个卷积块使用相同数量的输出通道。 然而，在前向传播中，我们将每个卷积块的输入和输出在通道维上连结。

由于每个稠密块都会带来通道数的增加，使用过多则会过于复杂化模型。 而过渡层可以用来控制模型复杂度。 它通过`1x1`卷积层来减小通道数，并使用步幅为2的平均汇聚层减半高和宽，从而进一步降低模型复杂度。

# 序列模型

> 序列模型是一类能够处理**输入数据存在时间或顺序依赖**的模型。它接收**序列数据**作为输入，按照序列的时间步或位置依次处理信息，并在输出端返回与输入顺序相关的结果。

- 序列数据

  > 由一系列有严格顺序关系的元素组成，元素之间的顺序不可随意改变，改变顺序会影响数据的意义。

- 特点

  - 顺序敏感（Order-sensitive）：模型必须考虑输入数据的排列顺序
  - 可能存在长期依赖（Long-term dependency）：较早的输入可能对后续输出有影响
  - 输入与输出的序列长度可能相同或不同（如机器翻译）

- 时间步

  > 时间步是序列模型处理数据的最小顺序单元，不一定代表一个固定的物理时间长度，而是代表在序列中一个位置的数据点。

## 自回归模型

> 对于序列模型而言，我们并无必要从最初的时间步开始输入数据，只需要满足某个长度为ɥ的时间跨度。此时的序列模型被称为自回归模型

# 注意力机制

## 双组件框架

> 在这个框架中，受试者基于*非自主性提示*和*自主性提示* 有选择地引导注意力的焦点。
>
> 非自住性提示即自物体的一些特殊性，自主性提示来自人本身

通常，参数化的全连接层， 甚至是非参数化的最大汇聚层或平均汇聚层聚焦于非自主性提示。因此“是否包含自主性提示”将注意力机制与全连接层或汇聚层区别开来。

在注意力机制中，自主性提示被称为*查询*（query）。给定任何查询，注意力机制通过*注意力汇聚*（attention pooling） 将选择引导至*感官输入*（sensory inputs，例如中间特征表示）。 这些感官输入被称为*值*（value）

![image-20250817102541577](D:\笔记\图片\image-20250817102541577.png)

- 原理

  在注意力机制（尤其是 **Transformer**）中，我们有三个矩阵：

  - **Query (Q)**：表示我们当前要“提问”的东西，即希望找到什么信息。
  - **Key (K)**：表示每条候选信息的标签，用来和 Query 进行匹配。
  - **Value (V)**：表示候选信息的真正内容，即要返回的信息。

  > 注意力机制的过程：

  1. 拿着 Query 去和所有 Key 做相似度匹配（通常是点积，得到权重）。
  2. 根据相似度对对应的 Value 加权求和。
  3. 得到结果，这就是注意力输出。

# GAN

> 生成对抗网络(Generative Adversarial Networks),由 **Ian Goodfellow** 在 2014 年提出的深度学习模型。

## 核心思想

通过 **博弈（对抗）** 的方式训练两个神经网络，一个网络被称为生成器（Generator, G），另一个网络称为判别器（Discriminator, D）。

- 生成器

  输入随机噪声（通常是高维向量，服从某种分布），输出“伪造”的数据样本。

  它的目标是生成看起来尽可能真实的数据，能够“欺骗”判别器。

- 判别器

  输入可以是真实数据，也可以是生成器生成的伪造数据，输出一个概率（判断输入是真实的还是伪造的）。

  它的目标是尽可能正确地区分 **真实数据** 与 **伪造数据**。

两个神经网络通过**零和博弈**不断对抗:判别器努力提高辨别真假的能力,生成器努力提高伪造的“以假乱真”能力。最终达到一个 **纳什均衡**：判别器无法轻易分辨真伪，生成器生成的样本与真实数据分布非常接近。

## 训练

### 训练目标

- **生成器 G**：输入随机噪声 z∼p(z)，生成样本 G(z)，目标是“以假乱真”，让判别器误以为它生成的数据是真实的。
- **判别器 D**：输入一张样本（可能是真实数据 x∼Pdata(x)，也可能是生成数据 G(z），输出一个概率 D(x)∈[0,1]，目标是尽可能正确地区分真（标签=1）和假（标签=0）。

所以，GAN的目标函数是一个 **minimax博弈**：

![image-20250818155145035](D:\笔记\图片\image-20250818155145035.png)

- 第 1 项：让判别器能正确识别真实样本（给真实样本的判别结果尽量接近 1）。
- 第 2 项：让判别器能正确识别生成样本（给伪造样本的判别结果尽量接近 0）。

![image-20250818155241491](D:\笔记\图片\image-20250818155241491.png)

- 生成器只关心“让判别器把假样本当成真样本”的能力。
- 所以它希望判别器对生成样本的输出 **D(G(z)) 接近 1**，从而使 log⁡(1−D(G(z))) 变小。

![image-20250818155320451](D:\笔记\图片\image-20250818155320451.png)

训练过程一般分为以下步骤：

1. **判别器训练（D）**

   - 从真实数据分布采样 x∼Pdata(x)

   - 从噪声分布采样 z∼p(z)，通过生成器得到 G(z)；

   - 判别器同时学习区分真实数据和生成数据，最大化：

     ![image-20250818153925679](D:\笔记\图片\image-20250818153925679.png)

2. **生成器训练（G）**

   - 生成器的目标是欺骗判别器，让 D(G(z))尽量接近 1；

   - 也就是最小化：

     ![image-20250818153947161](D:\笔记\图片\image-20250818153947161.png)

   - 在实践中，常用等价的“非饱和”形式来避免梯度消失问题：

3. **交替迭代**

   - 在一个 batch 里，先更新判别器参数一次（或几次），再更新生成器参数一次；
   - 这样 D 和 G 就在不断对抗中提升。

## 训练误差和泛化误差

> *训练误差*（training error）是指， 模型在训练数据集上计算得到的误差。 *泛化误差*（generalization error）是指， 模型应用在同样从原始样本的分布中抽取的无限多数据样本时，模型误差的期望。

## 过拟合/欠拟合

> 当模型在训练数据上拟合的比在潜在分布中更接近,此时称模型过拟合(overfitting),用于对抗过拟合的技术称为正则化（regularization）。

- 当训练误差与泛化误差都很严重，但它们之间仅有一点差距，且模型无法降低训练误差，这可能意味着模型过于简单，无法捕获试图学习的模式，这种现象称为欠拟合(underfitting)
- 当训练误差明显低于验证误差时，表明严重的过拟合。

## 权重衰减

> Weight Decay，是最广泛使用的正则化的技术之一。本质就是在损失函数中加入一个**惩罚项**（通常是权重的平方和）从而限制权重的绝对值不要太大，以防止模型在训练集上过拟合。

![image-20250809114101013](D:\笔记\图片\image-20250809114101013.png)

- L(w,b)是损失函数，`||W||²`为平方范数(L₂),λ为正则化常数，是一个非负超参数，除2是为了求导后与平方相消。
- L₂是矩阵中所有元素的平方和。

## 泛化

> 模型在**未见过的、与训练数据同分布的样本**上仍然能保持较好性能的能力称为泛化能力。泛化通常会受到下面几个因素的影响

1. 可调整参数的数量。当可调整参数的数量（有时称为*自由度*）很大时，模型往往更容易过拟合。
2. 参数采用的值。当权重的取值范围较大时，模型可能更容易过拟合。
3. 训练样本的数量。即使模型很简单，也很容易过拟合只包含一两个样本的数据集。而过拟合一个有数百万个样本的数据集则需要一个极其灵活的模型。

## 科学理论的可证伪性标准

> 由波普尔提出：一个好的科学理论，不仅能够与现有观测数据相符合，还必须存在具体的测试使其**有被证明为错误的可能性**。
>  换句话说，科学理论必须**既能解释现实，又能被现实检验甚至推翻**。

## 验证集

> 验证集用于训练过程中(每轮或每隔几轮)，用于调整模型的超参数、监控性能、防止过拟合。

- **调超参数（Hyperparameter tuning）**
   比如学习率、网络深度、正则化系数、批量大小等，不能直接在训练集上调这些，否则会让模型记住训练集的特征，失去泛化能力。

- **模型选择（Model selection）**
   如果你有多个候选模型（不同架构/不同超参组合），用验证集来比较谁的性能更好。

- **早停（Early stopping）**
   在训练过程中，验证集的性能先上升后下降时，说明开始过拟合训练集，此时可以提前停止训练。

### K折交叉验证

当训练数据稀缺时，我们甚至可能无法提供足够的数据来构成一个合适的验证集。 这个问题的一个流行的解决方案是采用K*折交叉验证*。 这里，原始训练数据被分成K个不重叠的子集。 然后执行K次模型训练和验证，每次在K-1个子集上进行训练， 并在剩余的一个子集（在该轮中没有用于训练的子集）上进行验证。 最后，通过对K次实验的结果取平均来估计训练和验证误差。

## 模型容量

> 模型的容量（capacity）**一般来说就是指模型表达和拟合复杂函数的能力**

- 参数越多，模型容量通常越大，能拟合更复杂的函数。
- 容量大的模型更容易记住训练数据，也更容易过拟合。
- 容量小的模型受限于参数较少，表达能力有限，容易欠拟合。

## 暂退法

> 暂退法在前向传播过程中，计算每一内部层的同时注入噪声，这已经成为训练神经网络正则化的常用技术。

![image-20250809123640200](D:\笔记\图片\image-20250809123640200.png)

- p为暂退概率
- 训练时，神经元以概率 1−P保留，概率 P被置为0（失活）。
- 保留下来的神经元的输出除以 1−P，即放大 1/(1-P),这样保证训练时神经元的期望输出不变。
- 测试时不做dropout，直接用全部神经元输出。
- 通常在靠近输入层的地方设置较低的暂退概率

## 梯度消失与梯度爆炸

> 当一个深层神经网络在进行反向传播时，梯度的计算常常涉及到多个权重矩阵的相乘（来自链式法则）。这种反复相乘可能导致梯度的数值 **指数级** 地增大或减小，从而引发以下问题：梯度爆炸（Gradient Exploding）:梯度在反向传播过程中不断放大，导致更新步幅过大,从而使参数在一次更新中发生极大变化，训练过程不稳定甚至发散。或者出现梯度消失（Gradient Vanishing）:梯度在反向传播过程中逐渐趋近于 0,从而使参数更新几乎没有变化，早期层的权重无法有效训练，网络难以收敛。

- 梯度消失

  > 如sigmoid函数，它只有在输入接近0时，梯度才比较正常，否则梯度就会消失，因此sigmoid函数基本被舍弃，选用更稳定的ReLU函数

## 批量规范化

> 批量规范化(Batch Normalization, BN,BatchNorm)就是在层间用当前批次的均值和方差对数据进行标准化处理，再用可学习的缩放参数和平移调整使其恢复到适合的分布，从而让训练更稳定、更快。是一种常见的神经网络正则化与加速训练的技术。
>
> 在应用批量规范化时，批量大小的选择可能比没有批量规范化时更重要。通常，批量规范化应用于全连接层或卷积层之后，激活函数之前。

![image-20250813154328921](D:\笔记\图片\image-20250813154328921.png)

- u和σ为当前批次的均值方差，用于对数据进行标准化

- β为偏移参数，γ为拉伸参数，它们是可学习参数。

------

**面对不同层的输出，BatchNorm会有不同的行为**

- 全连接层

  > 在全连接层中，BN  **对每个神经元的输出维度，在整个 batch 上做标准化**。并为每个特征(即每一列)引入一对平移参数和缩放参数。

  ![image-20250822103426910](D:\笔记\图片\image-20250822103426910.png)

  - j为输出的一列

- 卷积层

  > 对于卷积层而言，BN计算每个通道在 **batch 内所有样本 + 空间位置**上的均值和方差，每个通道都有自己的拉伸参数与偏移参数。

  ![image-20250822103417846](D:\笔记\图片\image-20250822103417846.png)

------

**BatchNorm在训练与推理阶段表现出不同的行为**

- 训练阶段

  > 用当前 mini-batch 的均值和方差进行归一化并更新平移参数和缩放参数。

- 推理阶段

  > 在训练阶段，批量规范化层会维护一个移动平均的均值和方差。由于推理阶段通常样本批次很小，如果以样本的均值和方差来进行规范化，可能会波动很大。因此推理时会使用这个移动平均的均值和方差。这样推理时就不需要遍历整个训练集来计算全局均值/方差。

  ![image-20250813161827200](D:\笔记\图片\image-20250813161827200.png)

  - `momentum` 是一个**平滑系数**,决定了新统计量和历史统计量的融合比例

    - **平滑更新**：避免 `running_mean` 和 `running_var` 受单个 batch 的剧烈波动影响。

    **控制记忆长度**：

    - **大 momentum（接近 1）** → 更依赖当前 batch 的统计量，更新快，但噪声大。

    - **小 momentum（接近 0）** → 更依赖历史统计量，更新慢，但更稳定。

### 功能

**缓解梯度消失/爆炸**

- 在深层网络里，数据在层与层之间传播时分布可能越来越偏移，称为 **内部协变量偏移（Internal Covariate Shift）**。
- BatchNorm 会把每层的输入分布保持在比较稳定的范围，避免梯度过大或过小。

**加快收敛速度**

- 输入分布更稳定 → 学习率可以设置更大，训练更快。

**有一定正则化效果**

- 因为在 mini-batch 内做统计，带来了一点噪声，类似于 dropout 的效果，可以减少过拟合。

**允许使用更深的网络**

- 解决训练难度，使得 ResNet、DenseNet 这些超深网络可行。
