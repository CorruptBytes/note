# 概念

> HTML 指的是超文本标记语言 (Hyper Text Markup Language) ，它是用来描述网页结构的一种语言。HTML 不是一种编程语言，而是一种标记语言 (markup language)。 标记语言就是一套标记标签 (markup tag)。  
>
> HTML页面也被称为HTML文档，后缀名必须是 .html 或 .htm，浏览器可以读取 HTML 文档，并以网页的形式显示出它们

## 超文本的含义

1. 它可以加入图片、声音、动画、多媒体等内容（超越了文本限制 ）。
2.  它还可以从一个文件跳转到另一个文件，与世界各地主机的文件连接（超级链接文本 ）。 

## 语法规范

- HTML 标签是由尖括号包围的关键词，例如 `<html>`。

- HTML 标签通常是成对出现的，例如 `<html>` 和 `</html>` ，我们称为双标签。标签对中的第一个标签是开始标签，第二个标签是结束标签。

- 有些特殊的标签必须是单个标签（极少情况），例如 `<br/>`,我们称为单标签。

- 双标签关系可以分为两类：包含关系和并列关系。

## HTML页面骨架

  ```html
  
  <!DOCTYPE html>
  
  <html lang="en">
  
  <head>
  
      <meta charset="UTF-8">
  
      <meta name="viewport" content="width=device-width, initial-scale=1.0">
  
    <title>Document</title>
  
  </head>
  
  <body>
  
  </body>
  
  </html>
  ```

| 标签名                    | 定义             | 说明                                                         |
| ------------------------- | ---------------- | ------------------------------------------------------------ |
| <html></html>             | HTML标签         | 页面中最大的标签,我们称为根标签                              |
| <head></head>             | 文档的头部       | 注意在head标签中我们必须要设置的标签是title                  |
| <titile></title>          | 文档的标题       | 让页面拥有一个属于自己的网页标题                             |
| <body></body>             | 文档的主体       | 元素包含文档的所有内容,页面内容基本都是放到body里面的        |
| <!DOCTYPE>                | 文档类型声明标签 | 文档类型声明，作用就是告诉浏览器使用哪种HTML版本来显示网页。 |
| lang                      | 语言种类         | 用来定义当前文档显示的语言。en定义语言为英语zh-CN定义语言为中文对于文档显示来说，定义成en的文档也可以显示中文，定义成zh-CN的文档也可以显示英文 |
| <meta charset=" UTF-8" /> | 字符集           | 在<head>标签内，可以通过<meta> 标签的 charset 属性来规定 HTML 文档应该使用哪种字符编码。charset 常用的值有：GB2312 、BIG5 、GBK 和 UTF-8，其中 UTF-8 也被称为万国码，基本包含了全世界所有国家需要用到的字符.必须要写的代码，否则可能引起乱码的情况 |

  > 注意:
  >
  > 1. `<!DOCTYPE> `声明位于文档中的最前面的位置，处于 <html> 标签之前。
  >2. ``<!DOCTYPE> ``不是一个 HTML 标签，它就是 文档类型声明标签。

# 常用标签

## 标题标签(`<h1>`  ~ `<h6>`)

> 标题标签会对包裹的文字加粗加大，使网页结构更具有语义性。
>
> HTML提供了6种不同等级的标题标签，各自加粗加大效果不同

```html
<h1>我是一级标题</h1>
```

## 段落标签(`<p>`)

> 段落标签用于定义段落，段落标签与其他标签之间存在空隙以将文字分段显示

```html
<p>我是一个段落标签</p>
```

> 特点:
>
> 1. 文本在一个段落中会根据浏览器窗口的大小自动换行。
> 2. 段落和段落之间保有垂直的间距。
> 3. 段落标签中只能内嵌文本级内容，不可以放置块级元素

## 换行标签(`</br>`)

> 换行标签用于将文字或行内元素以及行内块元素强行换行
>
> 在HTML中，浏览器会将整个浏览器窗口的一行内容填满才会换行，换行标签可以使其强制换行

```html
</br>
```

## 文本格式化标签(`<strong>` `<em>` `<del>` `<ins>` `<u>`)

> 文本格式化标签用于设置文字的特殊显示效果

```html
<!-- 加粗标签 -->
    <strong>我是加粗标签</strong>
    <b>我是加粗标签</b>
<!-- 倾斜标签 -->
     <em>我是倾斜标签</em>
     <i>我是倾斜标签</i>
 <!-- 删除线 -->
     <del>我是删除线标签</del>
     <s>我是删除线标签</s>
<!-- 下划线 -->
      <ins>我是下划线标签</ins>
      <u>我是下划线标签</u>
```



## `<div>`与`<span>`

> 这两个标签没有任何语义，它们就是一个盒子，用来装内容的。

```html
 <!-- div 是 division 的缩写，表示分割、分区。span 意为跨度、跨距。 -->
<div> 这是头部 </div>
<span> 今日价格 </span>
```

> 特点：
>
> 1. `<div>`标签用来布局，但是一行只能放一个`<div>`,即div标签一人独占一行。是一个大盒子
> 2. `<span>` 标签用来布局，一行上可以多个` <span>`。是一个小盒子

## 图片标签`<img>`

> 图片标签可以指定一个图像地址，用于在html页面中显示图像

```
<img src="图像URL" />
```

> src是`<img>`标签的必须属性，用于指定图像路径，可以是文件路径，也可以是url路径

## 超链接标签(`<a>`)

> 为标签包裹内容添加跳转链接

```html
<a href="跳转目标" target="目标窗口的弹出方式"> 文本或图像 </a>
```

> href是必须属性，指定跳转目标；traget指定新页面的打开方式，默认为“`_self`”（在当前窗口打开),可修改为"`_blank`"(在新窗口中打开)

### 链接分类

- 外部链接

  > 跳转到外部网站的链接

- 内部链接

  > 跳转到当前网站内部的其他页面的链接

- 空链接

  > 使用 `href="#"` 或 `href="javascript:void(0)"` 创建的链接，通常用于占位或者通过 JavaScript 后续处理。

- 下载链接

  > 指向一个文件地址，当点击时会触发文件下载

- 网页元素链接

  > 可以为图片、按钮、图标等元素添加 `<a>` 标签，实现点击跳转的效果。

- 锚点链接

  > 跳转到当前页面或其他页面的指定位置（锚点），一般使用 `#id` 形式，即根据标签id跳转。

# 小知识

## 超文本的发展

- 出生

  > 超文本标记语言最初由万维网之父蒂姆·伯纳斯-李（Tim Berners-Lee）设计，他最初的设想是通过链接的方式，在不同文档之间非线性跳转，即超文本。此时的超文本仅仅是文本结构 + 链接，内容只是纯粹的文字

- 向超媒体发展

  > 随着网络的发展，超文本逐渐向超媒体发展，强调结构 + 链接 + 媒体内容+交互体验。超媒体在本质上和超文本是一样的，只不过超文本技术在诞生的初期管理的对象是纯文本，所以叫做超文本。随着多媒体技术的兴起和发展，超文本技术的管理对象从纯文本扩展到多媒体，为强调管理对象的变化，就产生了超媒体这个词。如今的超文本标记语言虽然名字仍然是HTML，但他的功能早已超越了单纯文本含义。

- 总结

  > Tim Berners-Lee 起初设计 HTML 是为了描述和链接文本文档（超文本），而今天，HTML 已发展为支持富媒体表现、交互体验的“超媒体”核心语言。

## 超媒体

> 超媒体是一种采用非线性网状结构对块状多媒体信息（包括文本、图像、视频等）进行组织和管理的技术。超媒体在本质上和超文本是一样的，只不过超文本技术在诞生的初期管理的对象是纯文本，所以叫做超文本。随着多媒体技术的兴起和发展，超文本技术的管理对象从纯文本扩展到多媒体，为强调管理对象的变化，就产生了超媒体这个词。
>
> 在超媒体中，超链接理念得到延伸，可以被称为泛超链接机制，不再局限为超文本中文本跳转文本的静态跳转，而是扩展为“任意媒介跳转任意媒介”的互动体验

## HTML中标签与元素的区别

> 标签通常是`<>`闭合而成,本质上就是一种**语法符号**,告诉浏览器这里是什么类型的内容
>
> 元素是标签加内容这个整体

```
<h1>标题</h1>

<h1> 和 </h1>是标签
"标题" 是内容
<h1>标题</h1> 是元素
```

## HTML文档中的特殊符号

| 符号 | 表示方法 |
| :--: | :------: |
| `<`  |  `&lt;`  |
| `>`  |  `&gt;`  |
|      |          |

## AJAX

`Asynchronous JavaScript and XML`意为异步的 JavaScript 和 XML，是一套前后端交互的技术方案，简而言之就是使用 JavaScript 异步向服务器发送 HTTP 请求获取数据，从而达到局部刷新页面的效果。
