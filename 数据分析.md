### 索引与切片

  > 以下操作均是在二维数组上进行

  ```python
  x = np.arange(100).reshape((10,10))
  #取第二行
  x[2]
  #取第二行后的所有行(包括第二行)
  x[2:]
  #取任意多行
  x[[2,8,9]]
  #取第二行到第五行
  x[2:6] #第二个索引表示的行取不到
  #取列操作，这里表示所有行都取，但是每行只取第二列

  x[[1,2],[4,5]] #这几个点会组成一个新的一维数组，这里表示取(1,4)和(2,5)这两个点，前面代表要取的行，后面代表要取的列
  ```


- 根据条件修改

  ```python
  x[x < 10] = 10 #x<10会返回一个bool数组，会将bool数组为True的位置赋10。

  ```

  

## Numpy数组运算

- 加减乘除(`+`,`-`,`*`,`/`)

  > 加减乘除均是对应位置元素进行加减乘除，如果两个数组形状不同，在可以广播的情况下会进行广播，使两者变成形状相同的数组然后执行运算

  ```python
  x = np.arange(12)
  y = x + 2 #此时2会自动广播为与x形状相同的数组，然后做矩阵加法
  y1 = x + np.arrange(12,24) #同矩阵加法，对应位置相加
  ```

  > 注意:
  >
  > - numpy数组在广播时会从右向左对齐维度，缺失的维度会补1，即一维数组(4,)在广播时会解释为(1,4)然后进行广播，高维数组也是如此

- 除零

  > Numpy中，如果数组元素为0，除0后为nan(not a number),如果元素非0，除0后为inf(infinity,无穷)
  
- 比较运算符

  ```
  x = np.arange(10)
  x < 10 #会返回一个bool数组
  ```

- 求和

  ```
  np.sum(t,axis = 0) # 沿0轴求和，保留0轴
  np.sum(t) #整个数组求和
  t.sum()
  ```

- 求均值

  ```
  t.mean(axis = 0)
  ```

- 求中值

  ```
  np.median(t,axis =0 )
  ```

- 求最大值/最小值

  ```
  t.max(axis = 0) 
  t.min(axis = 0)
  ```

- 求极值

  > 极值为最大值与最小值的差

  ```
  np.ptp(t,axis = 0)
  ```

- 求标准差

  ```
  t.std(axis=0)
  ```

- 转置

  ```
  t.T #这个属性保存了转置后的数组
  t.transpose()
  t.swapaxes(1,0)
  ```

- 行列交换

  ```
  #行交换
  t[[1,2],:] = t[[2,1],:]
  #列交换
  t[:,[0,2]] = t[:,[2,0]]
  ```

## 小知识

### nan的注意点

```python
#将一个元素赋值成nan
x[2,2] = np.nan
#两个nan是不相等的
np.nan == np.nan #False
np.nan != np.nan #True
#由于nan不相等，可以通过下列方法判断数组中nan的个数
np.count_nonero(t != t) #判断数组中非0的个数 True被视为1，False视为0

#可以通过下面方法判断元素是否为nan并将其赋值
t[np.isnan(t)] = 0 #isnan()返回一个bool数组

#nan参与任何计算都等于nan
1 + nan = nan
```
